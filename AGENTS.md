# 美团智能取餐柜介绍网站

## 项目概述
美团智能取餐柜介绍网站，展示智能取餐柜的产品特性、使用流程、技术参数和合作加盟信息。

## 技术栈
- React 18 + TypeScript
- React Router DOM (HashRouter)
- Tailwind CSS
- Framer Motion (动画库)
- FontAwesome (图标库)
- Webpack 5

## 路由结构

| 路由路径 | 页面 | 说明 |
|---------|------|------|
| `/` | Home | 首页 - 产品概览、核心价值、应用场景 |
| `/features` | Features | 功能特性页 - 8大核心功能详解 |
| `/process` | Process | 使用流程页 - 3步取餐动画演示 |
| `/specs` | Specs | 技术参数页 - 硬件规格、柜体尺寸、配置方案 |
| `/contact` | Contact | 合作加盟页 - B端商户申请、C端用户反馈 |

## 页面功能

### 首页 (Home)
- Hero区域：品牌展示、核心卖点
- 数据统计：日均存取单量、保温温度、服务时间等
- 核心价值：无接触配送、餐品安全、物业管理、骑手效率、楼宇智能化
- 应用场景：写字楼、医院、酒店、产业园、商业中心、学校

### 功能特性页 (Features)
- 8大核心功能：扫码取餐、智能保温、紫外线消毒、24小时自助服务、智能通知、安全可靠、多平台支持、智能管理
- 核心优势：安全高效、灵活组合、强大功能、专业服务

### 使用流程页 (Process)
- 3步取餐流程：骑手存餐 → 自动通知 → 用户取餐
- 详细步骤动画演示
- 多种取餐方式说明

### 技术参数页 (Specs)
- 硬件规格：颜色、格口数、电压、功率、温度范围等
- 柜体尺寸：单柜尺寸、格口尺寸
- 配置方案：单机方案、标准方案、增强方案
- 安全认证：CCC认证、CE认证等

### 合作加盟页 (Contact)
- 合作模式：美团提供 vs 物业方提供
- 物业方收益
- 合作咨询表单（B端商户/C端用户）
- 成功案例展示
- 联系方式

## 设计规范

### 品牌色系
- 主色：美团黄 (#FCD34D / #FBBF24)
- 辅助色：橙色渐变
- 背景：白色、灰色系
- 文字：深灰色系

### 响应式设计
- 移动端优先
- 断点：sm(640px), md(768px), lg(1024px)

### 动画效果
- 使用 Framer Motion 实现
- 滚动触发动画
- 悬停交互效果

## 开发命令

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```
